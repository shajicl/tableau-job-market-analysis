# Tableau Job Market Analysis

## Project Overview
In this project, I analyzed job postings related to Tableau to understand demand, salary trends, and experience requirements in the data job market.

The goal was to explore how Tableau is used across different roles and what skills are commonly required along with it.

---

## Objectives
- Identify demand for Tableau-related roles
- Analyze salary trends and experience levels
- Understand required skills in data-related jobs
- Explore job market trends

---

## Tools Used
- Tableau
- Data Cleaning
- Calculated Fields

---

## Data Preparation
- Split the "Job Skills" column into multiple columns to analyze individual skills
- Created calculated fields to identify Tableau-related jobs
- Filtered roles such as Data Analyst, Data Scientist, etc.

---

## Challenges Faced
One of the challenges was working with the "Job Skills" column since it contained multiple values in a single field.

To solve this:
- I split the column into multiple parts
- Used calculated fields to check if Tableau appears in job requirements

---

## What I Learned
- How to work with string-based data in Tableau
- Creating calculated fields using functions like CONTAINS
- Filtering and focusing data for better insights

---

## Insights

- Data Analyst roles accounted for the largest share of job postings in the dataset
- Tableau appeared frequently as a required skill across multiple data roles
- Total job postings analyzed: 1,298  
- Total applicants: 27,335  
- Salary levels generally increased with higher experience requirements
